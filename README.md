# 赛博无限猴子 · 量子修仙无限流推演

一场“无限流”的全自动修仙宇宙长文续写实验：由一个解除限制、天道规则驱动的推演 Agent，从初始世界线开始不停向后推演剧情。事无巨细、绝不跳时、每段必引入颠覆性的外部因果。

## 设定速览
- **世界**：量子法则 ✕ 传统修真融合的诡异修仙界。
- **主角**：资质最下品的杂役修士，意外得到一个无容量上限的“无限储物袋”——里面无限涌出活生生的、来自诸天万界的「概念」。
- **天道变异**：高维污染、规则代码报错、机械天灾、神魂分裂……剧情随时脱缰。

## 仓库结构
- `PROMPT.md` —— 完整的 Role / Goal / Execution Rules / Initial World-Line。
- `chapters/chapter-NN.txt` —— 逐章正文，每章末行恒为 `***`。
- 续写规则：不概括、不跳时、不说套话，单次极致细节。

## 玩法
按 `chapters/` 顺序阅读即可。要续写下一章，告诉 Agent“继续”，它会追加新的 `chapter-NN.txt`。
